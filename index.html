📖 What is TFD OS?

TFD OS (TinyFoxyDOS) is a text-based operating system built from scratch by Sadman in 2026. It runs on real hardware using VGA text mode — no GPU needed.

This is a hobby project for learning low-level programming. It includes a command shell, text editor, and 3 classic games. Download the source code and build your own OS!

"Small OS, Big Dreams" — Sadman, 2026

📀 Quick Install

# 1. Download TFD-OS.zip from above
# 2. Extract the ZIP
# 3. Write os.iso to USB with Rufus (DD mode)
# 4. Boot from USB (Legacy/CSM mode)
# 5. Press ENTER at TFD logo
# 6. Type FXC GAMES to play!

# Or test in QEMU:
qemu-system-i386 -cdrom os.iso -m 32

🐛 Known Issues (Alpha)

  • Notepad: ESC key to exit not yet implemented
  • Shutdown: Countdown completes but doesn't power off
  • No mouse support — keyboard only
  • No file system — cannot save files
